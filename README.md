### CAT AI vs MOUSE AI
 
 It is a game, which simulates two Q-Learning Agents
 in an environment. 
 
 The Mouse agent's goal is to get to the cheese
 avoiding the Cat. The cat agent's goal is to catch the mouse 
 before it gets to the cheese.
